Slow app # Application

Unfortunately, it is usually impossible to write an application without making mistakes. It often happens that one fix causes another error to appear. Your task is to find the function that caused the slow loading of the application.

  1. Run the app on an emulator or your physical device.
  2. Find out which function is slowing down the loading of the application when you press the "Start" button.
  3. Assign the name of the slow function to the solution variable as the String type.
