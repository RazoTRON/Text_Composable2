Say my name # Application

Implement a composable SayMyName . This composable should display the text "Say my name" (without quotes) on the screen. Each part of this text should have a custom style configuration:

  • "Say": black color;
  • "My": semi-bold and black color;
  • "Name": extra-bold and red color;
