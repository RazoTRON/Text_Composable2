Application inspection # Application

It's time to put the acquired knowledge into practice. We have a small application running much slower than it should. The problem is that the application takes a long time to start when you press the "Start" button. To fix the problem, you must find out which function is slowing down the application launch.

  1. Run the app on an emulator or your physical device.
  2. Find out which function is slowing down the loading of the application when you press the "Start" button.
  3. Assign the name of the slow function to the solution variable as the String type.

[HINT] Luckily, the developers who messed up the code took care of you and added custom trace events. [/HINT]
